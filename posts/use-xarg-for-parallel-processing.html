Use the following command to print number of cores

cat /proc/cpuinfo | grep processor | wc -l


xargs takes "-P" flag to spawn a given numbers of processes.
It also takes "-n" to take a definite number of rows from the input to each process.

e.g. To uncompress a number of files parallelly

ls *.gz|xargs -P 4 -n 1 gunzip
Done!
